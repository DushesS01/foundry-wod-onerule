# WoD: 1s Cancel Successes + 10s Explode  
**Универсальный модуль для Foundry VTT**  
Автоматически превращает ВСЕ броски d10 в системе World of Darkness 20th Anniversary Edition в правильную классическую механику, без необходимости править шиты и формулы.

### Что делает модуль
- 10-ки взрываются (даже если в системе их нет)
- 1-ки отменяют успехи
- 0 успехов + есть 1-ки → БОТЧ!
- 0 успехов без 1-ок → Провал
- 5+ чистых успехов → ИСКЛЮЧИТЕЛЬНЫЙ!
- Работает на ЛЮБЫХ бросках d10 из шитов, предметов, макросов, чата
- Поддержка русского и английского языка (автоматически подстраивается под язык игрока)

### Установка
1. В Foundry VTT открой «Управление модулями» → «Установить модуль»  
2. Вставь Manifest URL:  
   ```
   https://raw.githubusercontent.com/DushesS01/foundry-wod-onerule/main/module.json
   ```
3. Нажми «Установить»  
4. Включи модуль в мире  
Готово. Ничего больше делать не нужно.

### Как пользоваться
Просто кидай проверки как обычно — из шита персонажа, из оружия, из дисциплин, из макросов.  
Модуль сам подхватит любой бросок d10 и в чате добавит строку:

```
Чистый результат: 3 успех(ов) (сложн. 6, −1 единица)
```
или
```
Чистый результат: БОТЧ! (сложн. 6, −4 единицы)
```

### Примеры работы
| Бросок из шита | Что покажет чат |
|------------------|-----------------|
| 7d10! (как в WoD20) | Правильные взрывы + отмена 1-ок |
| 5d10 (без восклицания) | Модуль сам посчитает успехи ≥6 и отменит 1-ки |
| 8d10cs>=8 (CoD-стиль) | Тоже полностью поддерживается |

### Настройки
Никаких. Модуль работает из коробки и не требует конфигурации.

### Совместимость
- Foundry VTT v11 – v13  
- World of Darkness 20th Anniversary Edition (официальная система)
