# WoD: 1-ки отменяют + взрывы 10

Мини-модуль для Foundry VTT.  
Делает ВСЁ автоматически:

- 10-ки взрываются
- 1-ки отменяют успехи
- 5+ чистых успехов = исключительный
- 0 успехов + есть 1-ки = ботч

Просто используй обычные формулы в шитах:
`@attributes.strength + @skills.athletics}d10x10cs>=6`
или
`8d10x10cs>=8` для Chronicles of Darkness

Ничего больше не надо настраивать. Работает везде.
